Allow your website visitors to post comment via their Facebook account

== Description ==

= Key Features =
* Title to display above Facebook Comments box
* Option to change text color
* Option to change font size
* Option to chnage title background color
* Option to change width size of facebook comment box
* Option to change colorscheme light and dark
* Option to increase or decrease numbers of comments
* Option to change Position of Comment Box
* Option to change languages of comments
* Option to change HTML tag of the title
* Option to integrate facebook comments with different page groups
* Customize target url for facebook comments box

= Benefits of facebook comments =
* Real names and identities greatly reduces the number of trolls and anonymous cowards in comments.
* Social virality boosts traffic by creating a feedback loop between Facebook and participating sites. Friends pull in their friends, creating a social entry point to your site.
* Automatic sign-in if you already signed into Facebook elsewhere, lowers the barriers to commenting.
* Most “liked” comments get voted to the top.  It also knows who your friends are, so you will see those comments first.

== Installation ==

= Installation =
* Download the plugin from this page
* Navigate to **Plugins** > **Add New** page in admin panel of your website
* Click **Upload Plugin**, upload the plugin zip and click **Install Now** button
* Activate the plugin when installation is complete

= Configuration =

After installing and activating the plugin, you will see **Amir Social Comments** in the left sidebar in your website admin panel.
